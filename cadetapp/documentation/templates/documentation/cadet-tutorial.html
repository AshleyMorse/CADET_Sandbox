User Tutorial

  1. The dashboard will be the first page the user is brought to when connecting
  2. User will choose the upload button located on the left banner of the page and a popup will appear to allow chosen files to be uploaded
  3. Under the "Options" menu, the user will be able to choose different configurations for the tool
  4. Under the "Distributions" menu there is the choice between "Topics Distribution" and "Instructor Distribution" for comment parsing
  5. After the tool has been run, there is the option of exporting the distributions into an excel document
